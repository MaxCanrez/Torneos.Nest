# Torneos Api

1. Levantar la base de datos 

```
docker compose up
```
2. Instalar librerias 
```
npm install
npm i -g @nestjs/cli
npm i class-validator class-transformer
```
3. Ejecutar el comando 
```
npm run start:dev
```
